## FAQ

**How do I make changes to the site?**  
Edit the static HTML content, commit the updates to the repository, and publish again if necessary.

**Where is this hosted?**  
The project is meant to be served through GitHub Pages.

**May I edit the content or page structure?**  
Yes. Since the site is static, the HTML, text, and links can be changed directly.

**What should I check if the page is not showing after publish?**  
Verify the Pages settings, make sure the correct source branch or folder is selected, and confirm the site files are committed.

**Do I need to install a separate application?**  
No. The published project is a single static page, so access happens in the browser.
